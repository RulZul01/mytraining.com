# Fourtitude Frontend Test

A responsive frontend implementation based on a design brief.

## 🔥 Features
- Fully responsive layout (desktop & mobile)
- Fixed header with smooth scrolling navigation
- Interactive cards with hover effects
- Form validation (email & phone format with error messages)
- Floating label inputs
- Success modal with overlay
- Subtle animations (fade-in, floating shapes, parallax)

## 🛠 Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## 🎯 Focus
This project focuses on:
- Clean UI/UX
- Code structure & readability
- Smooth user interactions
- Performance without external libraries
