# Fourtitude Frontend Test

This project is a frontend test implementation based on a provided design.

## Features
- Responsive layout (desktop & mobile)
- Fixed header with smooth scrolling
- Interactive cards with hover effects
- Form validation (email & phone format)
- Floating label inputs
- Success modal with overlay
- Clean UI with animations

## Tech Stack
- HTML
- CSS (Vanilla)
- JavaScript (Vanilla)

## Notes
- No external libraries were used as per requirements
- Focused on clean structure, UX, and performance
